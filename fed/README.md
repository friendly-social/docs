# Friendly Evolution Decisions

This folder contains a list of decisions that were made about Friendly future features. It does not contain any platform-specific things which should be addressed in a respective repositories. It is convenient to have a history of made decisions when you develop clients.

To initiate a new FED the Discussion is held first on 'Discussions' or in any other convenient for developers way. Calls are welcome as well. You do not start with PR.

## Naming

- `fed-number-slug.md`

Where `number` is incremented and slug is a code name for that feature.
